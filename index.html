üìã APIs to Be Tested

API 1: Get All Products List

URL: https://automationexercise.com/api/productsList
Method: GET
Response Code: 200
Response: All products list

API 2: Get All Brands List

URL: https://automationexercise.com/api/brandsList
Method: GET
Response Code: 200
Response: All brands list

API 3: POST To Search Product

URL: https://automationexercise.com/api/searchProduct
Method: POST
Parameter: search_product (e.g., top, tshirt, jean)
Response Code: 200
Response: Searched products list

API 4: POST Without Parameter

Response Code: 400
Message: Bad request, parameter missing

API 5: POST To Verify Login (valid details)

URL: https://automationexercise.com/api/verifyLogin
Method: POST
Parameters: email, password
Response Code: 200
Message: User exists!

API 6: POST To Verify Login (missing email)

Response Code: 400
Message: Bad request, email or password parameter is missing in POST request.

API 7: POST To Verify Login (invalid details)

Response Code: 404
Message: User not found!

API 8: POST To Create/Register User Account

URL: https://automationexercise.com/api/createAccount
Method: POST
Parameters: name, email, password, title, birth_date, birth_month, birth_year, firstname, lastname, company, address1, address2, country, zipcode, state, city, mobile_number
Response Code: 201
Message: User created!

API 9: DELETE To Delete User Account

URL: https://automationexercise.com/api/deleteAccount
Method: DELETE
Parameters: email, password
Response Code: 200
Message: Account deleted!

API 10: PUT To Update User Account

URL: https://automationexercise.com/api/updateAccount
Method: PUT
Parameters: name, email, password, title, birth_date, birth_month, birth_year, firstname, lastname, company, address1, address2, country, zipcode, state, city, mobile_number
Response Code: 200
Message: User updated!

API 11: GET User Account Detail by Email

URL: https://automationexercise.com/api/getUserDetailByEmail
Method: GET
Parameter: email
Response Code: 200
Response: User detail

API 12: DELETE To Verify Login

URL: https://automationexercise.com/api/verifyLogin
Method: DELETE
Response Code: 200
Message: Successfully.
